Описание проекта. Запуск:
- клонируем репу. 
- заходим в Frontend
- npm install
- npm run start
- заходим в Backend
- npm install
- npm run start:dev

Далее у нас появляется в браузере работающий Frontend на localhost:3000, и backend на localhost:5000. Политика CORS разрешена в настройках, поэтому все должно работать.
По адресу localhost:5000/api доступен SWAGGER, в котором можно посмотреть все ендпойнты и соответственно, поиграться с беком.
